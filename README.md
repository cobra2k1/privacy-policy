Privacy Policy
Effective Date: June 20, 2026

This Privacy Policy explains how our Discord security and moderation bot ("the Bot"), developed and operated by Cobra, collects, uses, and protects your data when you use it within Discord servers. By adding or interacting with the Bot, you agree to the terms outlined in this policy.

1. Data We Collect
To provide automated security, anti-nuke, and moderation features, the Bot requires access to certain information from the Discord API. The data we collect includes:

Discord IDs: User IDs, Guild (Server) IDs, Channel IDs, and Role IDs.

Message Content: The Bot briefly scans message content and custom status presences in real-time to detect malicious links, spam, and unauthorized invites.

Moderation History: Warning counts, kick/ban logs, and automated actions taken by the Bot.

2. How We Use Your Data
The collected data is strictly used for server security and moderation purposes:

To actively protect servers from raids, spam, and disruptive behavior.

To execute automated moderation rules configured by server administrators.

To maintain internal caches for bulk moderation commands and logging.

Message Content is NOT used to train machine learning or AI models.

3. Data Storage & Security
We take the security of your data seriously.

All necessary data is stored securely in an encrypted database (such as SQLite) hosted on our private web infrastructure.

We do not sell, rent, or share your data with any third parties.

Ephemeral data (like scanned text that does not violate any rules) is processed in memory and immediately discarded.

4. Data Retention
We adhere to data minimization principles. Stored activity and moderation data (API Data) is automatically cleared and retained for no longer than 30 days, unless a server administrator specifically clears it sooner.

5. User Rights & Data Deletion
Users and server administrators have the right to request the deletion of their data at any time.

Users: You can request the removal of your data by contacting our support team.

Administrators: You can remove your server's data by kicking the bot from your server, which will trigger an automated deletion protocol, or by using the designated data-clearing bot commands.
